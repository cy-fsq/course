## 1. Product Overview
个人页面网站，展示郑利娜老师的课程信息和个人简介
- 主要目的是提供一个静态页面，展示教师的课程信息，方便学生了解课程内容
- 目标用户是广东科学技术职业学院商学院的学生和教师

## 2. Core Features

### 2.1 User Roles
| Role | Registration Method | Core Permissions |
|------|---------------------|------------------|
| Visitor | 无需注册 | 浏览页面内容 |

### 2.2 Feature Module
1. **首页**：个人简介、课程列表、联系方式

### 2.3 Page Details
| Page Name | Module Name | Feature description |
|-----------|-------------|---------------------|
| 首页 | 个人简介 | 展示郑利娜老师的个人信息，包括所属学院、专业等 |
| 首页 | 课程列表 | 展示多个课程信息，包括课程名称、简介等，后续可补充详细内容 |
| 首页 | 联系方式 | 提供联系方式，方便学生与教师沟通 |

## 3. Core Process
用户访问网站 → 浏览个人简介 → 查看课程列表 → 了解课程信息

## 4. User Interface Design
### 4.1 Design Style
- 主色调：蓝色系 (#1a56db) 和白色 (#ffffff)
- 辅助色：浅灰色 (#f3f4f6) 和深灰色 (#374151)
- 按钮风格：圆角矩形，悬停效果
- 字体：无衬线字体，标题使用较大字号
- 布局风格：卡片式布局，清晰的信息层次
- 图标风格：简约现代的线性图标

### 4.2 Page Design Overview
| Page Name | Module Name | UI Elements |
|-----------|-------------|-------------|
| 首页 | 个人简介 | 顶部导航栏，个人照片，姓名，职位，所属学院专业，简短介绍 |
| 首页 | 课程列表 | 卡片式布局，每个课程包含课程名称、简介、图标 |
| 首页 | 联系方式 | 联系信息卡片，包含邮箱、电话等 |

### 4.3 Responsiveness
- 桌面优先设计，适配移动设备
- 响应式布局，在不同屏幕尺寸下自动调整
- 触摸优化，确保在移动设备上的良好体验

### 4.4 3D Scene Guidance
- 不适用，本项目为静态页面，无需3D场景
